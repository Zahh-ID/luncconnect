# Create React App Template + LUNCConnect

This is a simple LUNCConnect + [Create React App (TypeScript)](https://reactjs.org/docs/create-a-new-react-app.html) template for building Cosmos wallet-connected dApps on Terra Classic.

Get started quickly by simply running:

#### npm

```sh
npx create-react-app my-app --template luncconnect
```

#### yarn

```sh
yarn create react-app my-app --template luncconnect
```

#### pnpm

```sh
pnpm dlx create-react-app ./my-app --template luncconnect
```

---

This template includes:

- LUNCConnect for wallet connections
- Pre-configured Terra Classic (LUNC) chain
- Support for Keplr, Leap, and Galaxy Station wallets
- TypeScript for type safety
